    CSS

    CSS یا Cascading Style Sheets یک زبان برنامه‌نویسی نیست!‌ پس چیست؟ باید گفت CSS یک زبان استایل‌نویسی‌ست که با استفاده از آن ‌می‌توانید وبسایت‌های با ظاهر فوق‌ العاده ایجاد کنید. ما در دوره آموزش CSS سعی داریم که CSS را به شکل ساده و قدم به قدم و به شکل کامل به شما آموزش دهیم.
